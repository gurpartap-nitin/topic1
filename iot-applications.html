IoT Applications

Activity-1

  1. What is an IoT application?

    2. Full form of Iot is Internet of Things.Iot applications are applications which can share data between devices with the help of Internet, these applications require Iot devices.

  2. Can you think of some examples?

    • Smart devices
      • Wearables- apple watches,headphones
      • Home appliances- Tv,lights,fans
      • safety- smoke detectors
    • Emergency devices
      • Accident prevention devices
    • Home security devices

  3. What are the main features needed in an IoT application?

    • Reliability
    • scalability
    • security
    • compact nature of the devices
    • Connectivity
